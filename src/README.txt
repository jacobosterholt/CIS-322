This directory contains all of the files used to run the web app.

The files in this directory are:
	app.py - to be run with python; resposible for running the Flask app
	config.py - parses the json configuration file
	lost_config.json - configuration file for the web app
	templates - directory that contains html templates for some of the web pages
	-- create_user.html - displays the create new user screen
	-- login.html - displays the login screen
